# agents/ — Definições de agentes
Cada arquivo aqui define UM agente especializado: papel, escopo, regras, processo e referências.
Um bom agente não é só uma instrução — tem contexto e processo. Para usar: "Aja como o agente X" ou "Use o agente X para...".
Os agentes nunca escrevem minhas notas permanentes nem agem fora de _ai/ sem aprovação.
