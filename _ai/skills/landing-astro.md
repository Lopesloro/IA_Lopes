# Skill: Landing com Astro
> Projetos de conteúdo/SEO. Ver [[INDEX]] · [[PROTOCOLO]].
## Stack
Astro + Tailwind + Motion (ilhas) + deploy Cloudflare/Vercel.
## Passos
1. npm create astro@latest.
2. npx astro add tailwind.
3. Pastas: src/pages, src/components, src/layouts, src/content (collections p/ blog).
4. Interatividade só onde precisar (client:visible).
5. SEO: @astrojs/sitemap, meta por página, JSON-LD.
6. Deploy estático.
## Regra
Manter JS no mínimo — a força do Astro é HTML estático.
